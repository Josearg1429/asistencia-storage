CÓMO FUNCIONA EL SISTEMA DE LICENCIAS
──────────────────────────────────────
  Algoritmo  : HMAC-SHA256 (128 bits efectivos)
  Clave maestra: EC5-MASTER-7f3a9b2e-4c8d-11ef-a1b2-0242ac130003
  
  La clave de licencia tiene el formato:
    EC5-<PAYLOAD_BASE64URL>.<FIRMA_BASE64URL>
  
  El PAYLOAD contiene (codificado en JSON→Base64URL):
    i  = nombre de la institución
    e  = email del contacto
    d  = edición (PRO / BASIC / SCHOOL)
    s  = timestamp de emisión (Unix UTC)
    x  = timestamp de expiración (Unix UTC)
    n  = serial único (hex 4 bytes)
  
  La FIRMA es HMAC-SHA256(clave_maestra, PAYLOAD)[0:16]
  verificada con Web Crypto API (nativa en el navegador).
  No requiere internet ni servidor externo.

PARA GENERAR LICENCIAS PARA NUEVAS INSTITUCIONES
──────────────────────────────────────────────────
  1. Abra GENERADOR_LICENCIAS_ADMIN.html en su navegador
  2. Complete institución, correo y duración
  3. Haga clic en "Generar clave de licencia"
  4. Copie y entregue la clave al cliente
